# Web Shell Atlas

This atlas is the browser-viewable proof artifact for `ADR-006 Web Frontend Topology`.

It renders the generated `/Users/test/Code/taxat_/prototypes/analysis/web-shell-atlas/atlas_data.json` payload as:

- an overview of the selected two-deployable topology
- one lawful `CALM_SHELL` stage
- one lawful `CLIENT_PORTAL_SHELL` stage
- one lawful `GOVERNANCE_DENSITY_SHELL` stage
- a verification lab for continuity, focus return, stale/recovery posture, and reduced motion

## Refreshing the atlas

1. Run `python3 /Users/test/Code/taxat_/tools/analysis/build_web_frontend_topology_adr.py`
2. Open `/Users/test/Code/taxat_/prototypes/analysis/web-shell-atlas/index.html` through the repo web server or any static file server rooted at `/Users/test/Code/taxat_`

## Contract intent

- The visual system is intentionally light, quiet, and typography-led.
- The portal stays customer-safe by deployable boundary rather than runtime hope.
- Route switches preserve shell ownership and only change object context.
- Support panels return focus to the opener.
- Reduced motion preserves semantic state instead of relying on choreography.
